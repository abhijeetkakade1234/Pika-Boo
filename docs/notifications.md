# Notification Rules

## MVP

- Reminder trigger default: 5 minutes before event start
- User can switch the lead window between 1, 5, 10, 15, 30, and 60 minutes
- Poll interval: 60 seconds
- One reminder per event per trigger window
- No native Windows toast fallback in MVP
- No sound required in MVP

## Event Priority

MVP can treat all events the same.

Later:

- Meetings
- Personal
- Urgent
- Deadlines
- Birthdays
