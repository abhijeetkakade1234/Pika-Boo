# Tech Stack

## MVP Choice

- Electron
- React
- TypeScript

## Why

- Fastest path with familiar web skills
- Easy access to tray, startup, and window controls
- Plenty good for an MVP background app

## Supporting Choices

- CSS animations first
- Google APIs client or direct OAuth/Event API wiring
- Secure local token storage mechanism chosen during implementation
- Electron builder for packaging

## Avoid For Now

- Tauri migration before MVP exists
- Heavy animation libraries if CSS is enough
- Local database unless simple file/secure-store persistence stops being enough
